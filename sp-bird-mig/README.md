# Spring Peak Bird Migration  
Bird migration hotspots in the Grand River Coastal Corridor study area from April 20 to May 31, years 2003 to 2008.  
Software used: ArcGIS Pro  
Data source: <a href="https://gl.audubon.org" target="blank">Audubon Great Lakes</a>, Michigan Natural Features Inventory  
CRS: EPSG:3857
